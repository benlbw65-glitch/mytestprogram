# Contributing to Simple Python Calculator

Thank you for your interest in contributing to the Simple Python Calculator! Here are some guidelines to help you get started.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally:
   ```bash
   git clone https://github.com/YOUR_USERNAME/mytestprogram.git
   ```
3. **Create a branch** for your feature or bug fix:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes** in [calculator.py](file:///Users/byungwoolee/mytestprogram/calculator.py).
5. **Write tests** in [test_calculator.py](file:///Users/byungwoolee/mytestprogram/test_calculator.py) to cover your changes.
6. **Run the test suite** to ensure everything passes:
   ```bash
   python3 -m unittest test_calculator.py
   ```
7. **Commit your changes**:
   ```bash
   git commit -m "Add descriptive commit message"
   ```
8. **Push to your fork** and **submit a Pull Request**.

## Coding Standards

- Write clean, documented Python code.
- Follow [PEP 8](https://peps.python.org/pep-0008/) style guidelines.
- Always include corresponding unit tests for new functions.
